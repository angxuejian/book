## 数据的表示

1、 二机制与十进制互转

按权展开法

对10进制，从低位到高位，依次要乘以10^0，10^1，10^2，10^3等； 也就是 1，10，100，1000

对2进制，从低位到高位，依次要乘以2^0，2^1，2^2，2^3等； 也就是 1，2，4，8

如果有小数点，小数点后就为 2^-1，2^-2，2^-3等；

<br>
示例：

11011 = `1x2^4 + 1x2^3 + 0x2^2 + 1x2^1 + 1x2^0` = `16 + 8 + 0 + 2 + 1` = 27

`11011` = `27`
